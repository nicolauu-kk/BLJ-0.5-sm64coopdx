# BLJ 0.5

Projeto experimental de BLJ focado em consistência para prática de speedrun.

## Objetivo
Criar um método mais consistente para executar BLJ, principalmente em superfícies inclinadas como rampas e escadas.

## Observações
- Funciona melhor em rampas e escadas.
- FPS limitado pode aumentar a consistência
   (recomendado 30 FPS para simular o comportamento original, mas o mod suporta taxas acima disso).
- Projeto focado em treino.
- Compatibilidade: Testado com o modelo padrão. Deve funcionar com personagens customizados que mantêm a física original de long jump.
